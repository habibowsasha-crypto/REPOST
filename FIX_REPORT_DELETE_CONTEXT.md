# FIX REPORT - Delete Notice Context

Добавлено улучшенное уведомление об удалении поста.

## Что изменено

- В SQLite `message_map` добавлено поле `source_date`.
- При копировании нового поста бот сохраняет:
  - текст / подпись исходного поста;
  - дату исходного поста;
  - ID копии;
  - тему назначения.
- При удалении поста в источнике бот отправляет уведомление в нужную тему с контекстом:
  - время обнаружения удаления;
  - дата исходного поста;
  - текст удалённого поста;
  - ID поста-копии.

## Важно

Фикс полноценно работает для новых постов после redeploy. Для старых постов дата исходного поста могла не быть сохранена.
