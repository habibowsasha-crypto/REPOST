# Telegram Broadcast Manager SAFE

Это безопасная версия проекта для управляемой рассылки по своим/разрешённым Telegram-чатам и каналам.

## Что изменено

- Убрана логика "спама" и авто-долбёжки по всем чатам.
- Все действия доступны только `ADMIN_ID_LIST`.
- Чаты импортируются выключенными. Рассылка возможна только по белому списку `✅`.
- Перед запуском есть превью и подтверждение.
- Есть SAFE-лимиты: минимальный интервал, пауза между чатами, максимум целей, максимум сообщений/час.
- Нет авто-вступления в чужие группы.
- Нет рассылки в личные сообщения.
- Есть история отправок.
- Активные повторяющиеся рассылки восстанавливаются после рестарта, если база сохранена.

## Как пользоваться

1. Создай бота через @BotFather.
2. Получи `API_ID` и `API_HASH` на https://my.telegram.org/apps.
3. Заполни переменные окружения.
4. Запусти `python main.py`.
5. В Telegram напиши своему боту `/start`.
6. Добавь рабочий Telegram-аккаунт.
7. Импортируй чаты или добавь чат вручную.
8. Включи только нужные чаты в белый список.
9. Создай рассылку, проверь превью и подтверди запуск.

## Railway

На Railway обязательно добавь Volume и примонтируй его в `/data`, иначе база с сессиями может потеряться после redeploy.

Рекомендуемый `DB_PATH`:

```env
DB_PATH=/data/tg_broadcast_manager.db
MEDIA_DIR=/data/media
```

## Важно по безопасности

`DB_PATH` содержит Telethon-сессии подключённых аккаунтов. Не выкладывай базу в GitHub, не отправляй её третьим лицам и не показывай логи с секретами.
