TgBlaster DM AI v1.0.3 - Menu & Group Discovery

Что изменено

1. Главное меню
- Добавлена текстовая команда «Меню» (регистр не важен).
- Добавлена команда /menu.
- Добавлена callback-кнопка «🏠 Главное меню».
- Любая команда «Меню» сбрасывает незавершённые мастера ввода.

2. Единое сообщение интерфейса
- Все callback-кнопки теперь обновляют текущее сообщение через edit.
- При ошибке редактирования новое сообщение не создаётся.
- Исправлен конфликт старой дедупликации callback: ключ теперь основан на query_id.
- Исправлено пересечение callback-префиксов account_ и account_info_.

3. Обнаружение групп подключённого аккаунта
- Добавлена кнопка «🔎 Найти группы аккаунта».
- Бот получает группы и супергруппы из диалогов подключённого аккаунта.
- Публичные и закрытые группы отображаются в отдельном списке.
- Сохраняются title, username, access_hash, peer_type, доступность и права аккаунта.
- Группы не удаляются при повторной синхронизации: недоступные отмечаются отдельно.
- После подключения нового аккаунта сразу предлагается запустить поиск групп.

4. Ограничение автоматической активации
- В рабочий список автоматически попадают только группы, которыми подключённый аккаунт управляет
  (creator/admin).
- Остальные членства отображаются только для просмотра.
- DM-выбор и восстановление мониторинга используют только доступные управляемые группы.
- Для закрытых управляемых супергрупп используется сохранённый access_hash.

5. Совместимость
- Старый callback add_all_groups_<id> перенаправлен на новую безопасную синхронизацию.
- Новых Railway Variables не требуется.
