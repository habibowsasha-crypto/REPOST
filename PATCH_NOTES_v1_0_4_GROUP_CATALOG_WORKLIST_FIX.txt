TgBlaster DM AI v1.0.4 - Group catalog/work-list consistency fix

Исправлено:
- Ручное добавление писало в pre_groups, а «Мои группы» читало groups. Теперь общий каталог отображает pre_groups.
- Успешное ручное добавление больше не создаёт ложное впечатление, что группа уже привязана к аккаунту.
- Найденные группы стали кликабельными: открывается карточка с ID, типом, доступом и состоянием рабочего списка.
- Для доступных управляемых групп добавлены кнопки «Добавить/убрать из рабочего списка».
- Группы, где аккаунт только участник, явно помечаются как «только просмотр».
- При потере доступа или прав управления группа автоматически исключается из рабочего списка, но остаётся в истории discovery.
- Добавлены уникальный индекс groups(user_id, group_id), очистка старых дублей и индекс discovery.
- Удаление из общего каталога теперь работает с pre_groups, а не с рабочими привязками аккаунтов.
- Карточка аккаунта показывает количество найденных и доступных только для просмотра групп.

Не изменено:
- Автоматические DM-задачи используют только проверенные рабочие группы, которыми аккаунт управляет.
- Не добавлена автоматизация для чужих групп, где аккаунт является только участником.
